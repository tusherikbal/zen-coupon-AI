# zen-coupon-AI

A lightweight WordPress plugin that provides AI-assisted coupon creation and management for WooCommerce stores.

## Features
- Generate coupon ideas and descriptions using AI
- Create and apply coupons directly from the WordPress admin
- Simple settings to connect your AI service (API key)
- Small, privacy-focused footprint — no tracking

## Installation
1. Upload the `zen-coupon-AI` folder to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Open **ZenCoupon AI** under the WordPress admin menu to configure.

## Configuration
1. Go to the plugin settings page in the WordPress admin.
2. Enter your AI provider API key (if required) and save settings.
3. Adjust default coupon templates and behavior as needed.

## Usage
- Navigate to the ZenCoupon AI admin page to generate coupon content.
- Review and publish coupons — they integrate with WooCommerce coupon management.

## Troubleshooting
- Ensure WooCommerce is active and up to date.
- Check your API key and network connectivity if AI generation fails.
